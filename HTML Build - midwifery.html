How do I get started?
 
It’s not a problem if you’re no longer a registered midwifery nurse – returning to practice is straightforward and there’s more than one way you can do it:
 
If you’ve worked for us in the last five years, you might be able to return right away through readmission.
   
If you’ve been away for a while, you’ll need to update your skills through either a Test of Competence (ToC) or a return to practice course.
 
 
Remember, if you have any questions, please feel free to call us on 0345 60 60 655 or email advice@healthcareers.nhs.uk. One of the local return to practice leads will be able to walk you through the different routes to returning.
